Electric Vehicle Database/Blog

This is a project I worked on for about 3 weeks during my studies at Bottega University

The homepage is a simple React Carousel using hardcoded images. The system stored records of any user submitted electric vehicles, as well as giving the user new and up to date articles from the News API

Cars submitted would automatically appear in the parking garage stored in a Python/Flask database

Users have the ability to scrap any records that do not belong, as well as submitting them.

Technologies:
  • - Python
  • - React JS
  • - Postgres
  • - Flask
  • - SQLAlchemy
  • - Carousel
  • - Animations
  • - News API
